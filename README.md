## Descrição do projeto

  O projeto NotaDez consiste em uma aplicação web direcionada aos docentes para adicionar, remover, atualizar e calcular notas
  dos alunos cadastrados. Deste modo, o educador terá uma experiência mais eficiente durante esta parte de seu trabalho.


## Stack
- **Backend:** [Node.js](https://nodejs.org/) com **TypeScript**
- **Frontend:**  
  - HTML5  
  - CSS3 com **Bootstrap**  
- **Banco de Dados:** [MySQL](https://www.mysql.com)  
- **Gerenciamento de Projeto:** [GitHub Projects](https://github.com/users/zarpela/projects/2) 

## Rodando o projeto

1. Clone este repositório com `git clone https://github.com/zarpela/ES-PI2-2025-T3-G03.git` ou crie um repositório manualmente e copie os arquivos.  
2. Acesse a pasta do projeto pelo terminal.  
3. Instale as dependências necessárias executando `npm install`.  
4. Rode o arquivo `NotaDezMySQL.sql` com seu SGBD que preferir.
5. Colocar a `.env` com as secrets necessárias na root do projeto.
6. Inicie o servidor em modo de desenvolvimento com `npm run dev`. 
7. Abra o navegador e acesse `http://localhost:3000` (a porta padrão utilizada é a 3000).
8. Pronto!
